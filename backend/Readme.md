## Installation Guides

* Install Go 

    [Go Installation Link Here!](https://go.dev/doc/install)

* Clone This Repository


* Add Value To ".env.example" File >> Change ".env.example" To ".env"
    You Must Add:
    1. Host
    2. Port 

* Run The Application With Your Terminal

    ```
    go run main.go
    ```
