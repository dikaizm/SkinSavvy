### Tentang App
Aplikasi ini menggabungkan teknologi kecerdasan buatan dengan pengetahuan dermatologi untuk memberikan solusi produk skincare yang dipersonalisasi sesuai dengan kebutuhan kulit masing-masing pengguna.

### Tech Stack
- Backend: Golang
- LLM: Google Gemini
- CV: Yolov8
- Frontend: Flutter
